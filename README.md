# 🚀 RetailPulse: Customer Analytics & Forecasting Dashboard

RetailPulse is an end-to-end data analytics project that transforms raw retail transaction data into actionable business insights using machine learning and interactive visualization.

---

## 🔍 Problem Statement

Retail businesses struggle to:

* Identify high-value customers
* Predict future demand
* Reduce customer churn

This project solves these problems using data-driven techniques.

---

## ⚙️ Features

* 📊 Data Cleaning & Preprocessing Pipeline
* 👥 Customer Segmentation (RFM + KMeans)
* 🔮 Sales Forecasting (Prophet)
* 📈 Interactive Dashboard (Streamlit)
* 💡 Automated Business Insights

---

## 🧠 Key Insights

* Revenue is concentrated in a small group of customers
* A large segment of customers is at risk of churn
* Forecasting achieves ~7% MAPE (high accuracy)
